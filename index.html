The purpose of the project

The User Business or Background of Project Effort The Client is requesting a web-based application for all the people who want to submit their papers for conferences. The user being an organizer should be able to advertise their event in the upcoming events list with the approval of the chair person (super admin but not a technical admin).

Goals of The Project
The primary goal of the project is to develop website that will allow people to submit paper in the conference. Motivation : To develop the web-based application that will provide good User Interface that provides user a user-friendly web application.
